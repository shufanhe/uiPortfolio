PROBLEM

On our university campus, some elevators restrict access to certain floors using card authentication. While this improves safety, it introduces usability challenges: students unfamiliar with the system often attempt to press a floor button without swiping their ID card first, leading to confusion and delays. The current interface provides little guidance.

SOLUTION

I conducted user research to identify friction points in the elevator experience, particularly for first-time users. The goal was to lay foundation for designing a clearer, more intuitive interface that maintains security while reducing confusion—especially in shared study or public access buildings like libraries.
